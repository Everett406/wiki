# 航空电子设备维修学院

## 📖 学院简介

航空电子设备维修学院（简称"电子学院"）是学校四个专业二级学院之一，现有在校学生近 **3000人**，其中陆军、海军、空军、火箭军等军种定向军士 **2000余人**。

学院开设有 **5个专业**，构成飞机电子设备维修专业群，该专业群是湖南省"楚怡"高水平高职专业群。

## 🎓 专业设置

### 1. 飞机电子设备维修

**培养目标**：培养具有航空电子设备的安装、调试、检测、维护能力，能够从事航空电子设备维修，航空电子产品的工艺实施、生产组织、技术管理等工作的技术技能人才。

**主干课程**：电工技术与应用、航空电子电气标准施工、电子电路分析与实践、高频电子技术与应用、航空电子综合显示系统、雷达与导航技术、飞机结构与系统等。

**就业方向**：航空装备制造和军航、民航、通航航空装备修理企业。

### 2. 民航通信技术

**培养目标**：培养具备航空通信技术的专业基础知识和技能，能从事飞机机载通信导航与雷达设备的制造与维修、空管通信导航与雷达管理、通信网络优化与管理等工作的高素质技术技能型人才。

**主干课程**：电路基础、低频电子技术、高频电子技术、脉冲数字电子技术、微处理器技术应用、通信原理与技术、卫星通信技术、雷达原理与设备、飞机无线电导航技术与设备等。

### 3. 导弹维修技术

**培养目标**：培养具有导弹维修必备的基础理论知识和专业知识，具有扎实的导弹修理、检测和装配能力，能够从事导弹装配、测试与维修、质量检验、技术管理等工作的高素质技术技能型人才。

**主干课程**：模拟电子技术、数字电子技术、高频电子线路、自动控制技术、导弹制导与控制系统、导弹引战系统、导弹构造、导弹检测技术、导弹测试系统等。

### 4. 应用电子技术

**培养目标**：培养具有较强的PCB设计和制作能力，电子产品和设备安装、调试、检验、生产、维修及管理能力的高素质技术技能型人才。

**主干课程**：电子电路分析与实践、高频电子技术、传感器与检测技术、PLC应用技术、单片机应用与实践、ARM技术与应用、电子产品设计与制作等。

### 5. 电气自动化技术

**培养目标**：培养具有较强电子电气设备及其控制系统的设计、安装调试和维护维修能力，能够从事航空电气设备与系统、电气自动化生产设备及控制系统的设计、安装调试、维护等工作的高素质技术技能型人才。

**主干课程**：电工技术、模拟电子技术、数字电子技术、电机与电力拖动、工厂电气控制设备、PLC应用与实践、变流与变频技术、供配电技术、单片机应用与实践等。
