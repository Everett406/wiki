# 航空服务与管理学院

## 📖 学院简介

航空服务与管理学院（简称"管理学院"）是学校四个专业二级学院之一，是国家级骨干专业教师培训点、湖南省货运代理项目职业技能竞赛（集训）基地。

学院开设有 **5个专业**，构成民航运输服务专业群。其中空中乘务专业是全国职业教育示范专业点、国家级骨干专业、全国职业教育交通运输类示范专业点、教育部空中乘务专业1+X证书试点单位。

## 🎓 专业设置

### 1. 空中乘务 ⭐国家级骨干专业

**培养目标**：培养掌握空中乘务必备的专业知识与技能，具有良好职业素养，能够从事空中乘务服务、机场地面服务等工作的高素质技术技能人才。

**主要课程**：客舱服务技巧、客舱安全管理、航空急救、乘务英语、航空服务礼仪、形体训练、航空卫生与保健等。

**就业方向**：国内外航空公司客舱乘务员、机场地面服务人员等。

### 2. 机场运行服务与管理

**培养目标**：培养掌握机场运行管理与服务必备的专业知识与技能，能够从事机场运行指挥、旅客服务、安全检查等工作的高素质技术技能人才。

**主要课程**：机场运营管理、机场场道管理、机场安全管理、旅客服务技能、航空运输地理等。

**就业方向**：民用运输机场、通用机场的运行管理与服务岗位。

### 3. 民航安全技术与管理

**培养目标**：培养掌握民航安全技术与管理必备的专业知识与技能，能够从事航空安全管理、安全检查、安全监察等工作的高素质技术技能人才。

**主要课程**：航空安全法律法规、航空安全管理、安检业务实务、危险品运输、应急处置等。

**就业方向**：航空公司、机场的安全管理部门、安检站等。

### 4. 民航运输服务

**培养目标**：培养掌握民航旅客运输、货物运输必备的专业知识与技能，能够从事旅客服务、票务销售、货运代理等工作的高素质技术技能人才。

**主要课程**：民航旅客运输、民航货物运输、航空运输地理、票务销售技巧、民航英语等。

**就业方向**：航空公司、机场、民航代理企业的运输服务岗位。

### 5. 通用航空航务技术

**培养目标**：培养掌握通用航空运行管理必备的专业知识与技能，能够从事通用航空运行协调、飞行指挥、航务管理等工作的技术技能人才。

**主要课程**：通用航空概论、航空气象、飞行程序设计、航行情报服务、通航运营管理等。

**就业方向**：通用航空公司、通用航空机场的航务管理岗位。
