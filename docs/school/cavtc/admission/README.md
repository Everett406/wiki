# 招生信息

## 📋 2025年招生章程

### 一、学校概况

长沙航空职业技术学院是全军唯一一所国民教育性质的普通高校，隶属中国人民解放军空军装备部，是国家"双高计划"建设单位、国家优质专科高等职业院校。

### 二、招生专业与计划

2025年学校设 **23个专业**，紧密对接航空产业，面向全国招生。

## 🎖️ 定向培养军士招生

### 招生对象

参加全国普通高校招生统一考试的应届高中毕业生，需通过政治考核、体格检查和面试。

### 2025年招生军种及专业

| 军种 | 招生专业 |
|------|---------|
| 空军 | 飞行器维修技术、飞机电子设备维修、导弹维修技术 |
| 海军 | 飞机电子设备维修、导弹维修技术 |
| 火箭军 | 应用电子技术、电气自动化技术、机电一体化技术 |
| 武警部队 | 飞行器维修技术、航空发动机维修技术、无人机应用技术 |
| 陆军 | 航空发动机维修技术、无人机应用技术、通用航空器维修 |

### 报考流程

1. **填报志愿**：在专科提前批填报长沙航空职业技术学院定向培养军士专业
2. **政治考核**：按要求参加政治考核
3. **体格检查**：参加军队组织的体格检查
4. **面试**：参加招生面试
5. **录取**：按成绩择优录取
