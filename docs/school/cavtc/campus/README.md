## 🏃 体育竞技

### 三大球成绩

长沙航院是湖南省高职院校体育强校，在三大球项目上表现突出：

| 项目 | 成绩 |
|------|------|
| 男子排球 | 🥇 **2025年湖南省大学生排球比赛（高职高专组）冠军** |
| 男子足球 | 🥉 2023年湖南省大学生足球联赛**亚军** |
| 男子足球 | 🥉 2025年湖南省大学生足球联赛**季军** |
| 篮球 | 稳居湖南高职组前八 |

### 运动会

学校每年举办学生田径运动会，已连续举办 **27届**：

- 🎪 参赛规模：676名体育健儿参赛
- 🏃 比赛项目：田径多项比赛
- 🎭 特色表演：舞龙、舞蹈、航模表演等

### 青年志愿者协会

**士官学院青年志愿者协会** 成立于2016年6月，是由志愿从事社会公益事业与社会保障事业的学生组成的公益性团体。主要活动包括：

- 🤝 社区志愿服务
- 🚗 交通协管
- 📚 支教活动
- 🏥 医院导诊

### 特色活动

| 活动类型 | 活动名称 | 描述 |
|---------|---------|------|

### 典型活动案例

#### 拳击技巧分享活动

机电学院2025级飞行器维修专业学生在跳马校区体育馆举办"'拳'释青春，筑梦航修"拳击技巧分享活动，丰富同学们课余生活，引导青年学子强健体魄。

#### 校园足球联赛

学院学生足球协会参加第四届星城学府足球联赛，学院足球队是湖南省大学生足球赛事的传统劲旅。
