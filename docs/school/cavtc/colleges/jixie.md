# 航空机械制造学院

## 📖 学院简介

航空机械制造学院是学校四个专业二级学院之一，开设有 **6个专业**，构成航空机械制造专业群。学院以培养航空制造领域高素质技术技能人才为目标，对接航空制造产业，服务航空装备制造与修理。

## 🎓 专业设置

### 1. 飞行器数字化制造技术 ⭐全国排名第一

**培养目标**：培养具有飞行器数字化制造方面的知识和技术技能，面向飞机制造行业的飞机装配、飞行器制造工程技术等职业群，能够从事飞机结构件加工、飞机钣金成形、飞机铆接装配、飞机装配与调试等职业岗位工作的高素质技术技能人才。

**主要课程**：航空零部件制造工艺、飞机装配工艺装备、数控编程与加工、飞机钣金技术、飞机铆接装配技术、飞机数字化装配与调试、航空零部件3D打印等。

### 2. 航空发动机制造技术

**培养目标**：培养具有航空发动机制造与装配等方面的知识和技术技能，面向航空、航天器及设备制造行业的航空发动机装配工、航空动力装置制造工程技术人员职业群，能够从事航空发动机零部件制造、检验及装配工作的高素质技术技能人才。

**主要课程**：航空发动机原理与构造、计算机辅助制造、多轴数控加工技术、特种加工、航空发动机制造工艺、航空发动机装配工艺与实践等。

### 3. 机电一体化技术

**培养目标**：培养具有机械加工、机电液气系统装调、机电产品维护维修等方面的知识和技术技能，面向航空装备、通用设备制造和专用设备制造业，从事机电一体化设备安装与调试、维修、自动生产线运维、工业机器人应用等职业岗位工作的高素质技术技能人才。

**主要课程**：机械制图、电气控制技术、PLC应用技术、机床电气系统装调与检修、液压与气动技术、机械结构装调与检修、工业机器人离线编程与仿真等。

### 4. 数控技术

**培养目标**：培养具有机械制图、机械加工工艺、数控编程、数控机床操作等方面的知识和技术技能，能够从事数控设备操作、机械加工工艺编制与实施、数控程序编制、质量检验等职业岗位工作的高素质技术技能型人才。

**主要课程**：机械制图与CAD、金属材料与热处理、计算机辅助设计、数控手工编程技术、数控加工工艺、多轴数控加工技术、数控加工技能实训、精密测量技术等。

### 5. 智能焊接技术

**培养目标**：培养具有焊接材料、焊接工艺、焊接工装设计、焊接设备使用、焊接机器人操作与维护等方面的知识和技术技能，面向航空工业和航空发动机制造、航空航天器修理等产业，从事常规或特种材料焊接的高素质技术技能型人才。

**主要课程**：先进焊接技术与设备、航空金属材料焊接、典型焊接结构与工艺编制、工业机器人应用编程、焊接机器人操作与维护、机器人焊接实训等。

### 6. 工业机器人技术

**培养目标**：培养具有电工电子、可编程控制、传感与检测、工业机器人操作编程、调试维修和系统集成等方面的知识和技术技能，面向航空航天、汽车制造、工程机械、船舶等产业，能够从事工业机器人的安装、编程、调试、维修、运行和管理等方面工作的高素质技术技能型人才。

**主要课程**：机械制图、电工与电子技术、液压与气动技术、传感器与检测技术、PLC应用技术、工业机器人离线编程与仿真、工业机器人现场编程、工业机器人应用系统集成等。

## 🏭 实训条件

学院建有完善的校内实训基地，配备有数控加工中心、焊接机器人、三坐标测量机等先进设备，为学生提供真实的职业环境训练。

## 💼 就业情况

毕业生就业方向主要面向：

- ✈️ 航空制造企业
- 🏭 航空航天器修理行业
- 🏗️ 通用设备制造企业
- 🤖 智能制造企业
