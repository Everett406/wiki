# HTML & CSS

## HTML 基础

HTML（超文本标记语言）是构建网页的基础。

### 常用标签

```html
<!-- 标题 -->
<h1>一级标题</h1>
<h2>二级标题</h2>

<!-- 段落与链接 -->
<p>这是一个段落</p>
<a href="https://example.com">链接</a>

<!-- 列表 -->
<ul>
  <li>无序列表项</li>
</ul>
<ol>
  <li>有序列表项</li>
</ol>
```

## CSS 基础

CSS（层叠样式表）用于控制网页的样式。

### 选择器

```css
/* 元素选择器 */
p {
  color: #333;
}

/* 类选择器 */
.container {
  max-width: 1200px;
  margin: 0 auto;
}

/* ID 选择器 */
#header {
  background-color: #2196f3;
}
```

### Flexbox 布局

```css
.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 16px;
}
```
