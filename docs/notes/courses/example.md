# 示例课程笔记

## 🎓 课程名称：计算机网络基础

### 第一讲：网络概述

#### 核心概念

- **OSI 七层模型**：物理层 → 数据链路层 → 网络层 → 传输层 → 会话层 → 表示层 → 应用层
- **TCP/IP 四层模型**：网络接口层 → 网际层 → 传输层 → 应用层

#### TCP vs UDP

| 特性 | TCP | UDP |
|------|-----|-----|
| 连接 | 面向连接 | 无连接 |
| 可靠性 | 可靠 | 不可靠 |
| 速度 | 较慢 | 较快 |
| 用途 | HTTP、FTP | DNS、视频流 |

### 第二讲：HTTP 协议

#### HTTP 请求方法

- `GET` - 获取资源
- `POST` - 提交数据
- `PUT` - 更新资源
- `DELETE` - 删除资源

#### 状态码

- `2xx` - 成功
- `3xx` - 重定向
- `4xx` - 客户端错误
- `5xx` - 服务器错误
