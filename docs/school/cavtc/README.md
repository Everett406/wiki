## 🏛️ 学校简介

长沙航空职业技术学院是全军唯一一所国民教育性质的普通高校，也是军队定向培养军士的定点高校。学校实行"军地双重管理、一体办学"体制，是：

- 🏆 **国家"双高计划"建设单位**（第三类高水平专业群 B 档）
- 🏆 **国家优质专科高等职业院校**
- 🏆 **湖南省楚怡高水平高职学校 A 档建设单位**
- 🏆 **国家级示范性职业教育集团**
- 🏆 **全国职业教育先进单位**
- 🏆 **全国教育系统先进集体**
- 🏆 **国家技能人才培育突出贡献奖**获奖单位
- 🏆 **全国黄炎培职业教育优秀学校奖**获奖单位

学校以航空产业为核心，构建了航空维修、航空电子、航空制造、航空服务四大特色专业群，全面覆盖航空装备保障、航空制造、民航服务、低空经济等全链条领域。

## 📚 院系设置

学校实行学院、二级学院（部）两级管理体制，设有 **8个教学院部**，开设 **25个专科专业**：

| 序号 | 二级学院（部） | 专业数量 | 主要专业方向 |
|:----:|---------------|:--------:|-------------|
| 1 | [航空机电设备维修学院](./colleges/jidian/) | 6个 | 飞行器维修、航空发动机、无人机等 |
| 2 | [航空电子设备维修学院](./colleges/dianzi/) | 5个 | 飞机电子设备维修、民航通信、导弹维修等 |
| 3 | [航空机械制造学院](./colleges/jixie/) | 6个 | 飞行器数字化制造、航空发动机制造等 |
| 4 | [航空服务与管理学院](./colleges/fuwu/) | 5个 | 空中乘务、民航运输、机场服务等 |
| 5 | [军士教育管理学院](./colleges/junshi/) | 7个 | 定向培养军士（空军、海军、火箭军、武警） |
| 6 | 马克思主义学院 | - | 思想政治教育 |
| 7 | 基础教育学院 | - | 基础课程教学 |
| 8 | 体育工作（教学）部 | - | 体育教学与竞赛 |
