# 学校历史沿革

## 📜 历史沿革

### 1973年 —— 创立

长沙航空职业技术学院创建于 **1973年**，前身为 **"中国人民解放军第五七一二工厂技工学校"**。

### 1982年 —— 中专时期

1982年改制为 **普通中等专业学校**。

### 1994年 —— 快速发展

1994年9月，更名为 **"中国人民解放军空军长沙航空工程学校"**。

同时，空军第二职工大学（原名"空军航空修理第二职工大学"，1983年2月创建）从成都迁入长沙。

### 1998年 —— 升格高职

1998年，学校与空军第二职工大学合并，升格为 **全日制普通高等职业学院**，即 **长沙航空职业技术学院**。

军队校名为 **"中国人民解放军空军航空维修技术学院"**。

### 2002年 —— 资源整合

原 **长沙工业学校** 并入学院，进一步扩大了办学规模。

### 2014年 —— 军士培养

2014年5月，学院成为 **二炮（现火箭军）定向培养直招士官** 在湘唯一试点院校。

### 2019年 —— 双高计划

2019年12月，被教育部、财政部列入 **第三类高水平专业群建设单位（B档）**。

### 2023年 —— 持续发展

2023年9月，学校入选 **第二批全国学校急救教育试点学校**。

2023年12月，被批准为 **2022年国家级高技能人才培训基地项目单位**。

### 2025年 —— 新征程

2025年，成功立项国家 **第二期"双高计划"** 建设项目，顺利通过湖南省高等职业院校办学能力评价。

## 🎯 办学定位演变

1. **1973-1982**：技工培训，为军工企业培养技术工人
2. **1982-1998**：中等专业教育，培养中等技术人才
3. **1998至今**：高等职业教育，培养高素质技术技能人才
4. **2014至今**：军民融合，定向培养军队专业技术军士
