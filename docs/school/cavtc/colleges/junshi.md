# 军士教育管理学院

## 📖 学院简介

军士教育管理学院（又称"定向培养学院"）是长沙航空职业技术学院承担军队定向培养军士任务的重要教学单位。学院是全军定向培养军士定点高校的重要组成部分，为空军、海军、火箭军、武警等多个军种培养专业技术军士。

## 🎖️ 办学特色

### 双重身份

定向培养军士生具有**双重身份**：
- 🎓 **学生身份**：在学院接受系统的专业教育
- 🎖️ **准军人身份**：享受军事化管理，接受军事训练

### 培养模式

采用" **2.5+0.5** "培养模式：
- 前2.5年：在学校完成理论课程和专业技能学习
- 后0.5年：在部队进行实习实训

## 📋 报考条件

### 基本条件

1. **招生对象**：参加全国普通高校招生统一考试的应届高中毕业生
2. **政治条件**：政治考核合格，思想政治素质好
3. **身体条件**：符合军队院校招收学员体格检查标准
4. **性别要求**：大部分专业只招收男生（具体以招生计划为准）

### 体格检查要求

- 身高：男性身高≥162cm（部分专业≥160cm）
- 视力：裸眼视力≥4.5（部分专业要求更高）
- 体重：符合军队院校招收学员体格标准
- 其他：无传染病、精神病史等

## 📚 优惠政策

### 学费补偿

定向培养军士享受**学费补偿或国家助学贷款代偿**，具体标准按国家相关规定执行。

### 生活待遇

- 🏠 提供统一住宿（军营式管理）
- 🎖️ 发放军装和军事训练用品
- 🍽️ 享受部队生活补贴
- 🏥 享受医疗保障

## 🎖️ 毕业后发展

### 待遇保障

军士毕业生待遇参照军队同等军衔军士标准执行：
- 💰 享受军士工资待遇
- 🏠 部队提供住房保障
- 🏥 享受免费医疗
- 🎖️ 服役满一定年限可转业安置

### 职业发展

- 📈 表现优秀者可晋升为高级军士
- 🎓 可参加军队院校深造
- 🏆 优先参加各类军事比武和任务

## 📊 就业情况

定向培养军士相关专业毕业去向落实率接近 **100%**，毕业后直接补充到部队相应军衔的军士岗位。
