# 航空机电设备维修学院

## 📖 学院简介

航空机电设备维修学院（简称"机电学院"）是全国职业教育先进单位、全国教育系统先进集体，荣获湖南省直单位五一劳动奖状，是全国航空工业职业教育教学指导委员会飞行器维修技术专业教学指导委员会主任委员单位。2024年航空机电设备维修学院学生党支部获评**全国第四批党建工作样板支部**。

机电学院组建的飞行器维修技术专业群为中国特色高水平高职院校高水平专业群建设项目，同时是湖南省示范性特色专业群，入围湖南省楚怡高水平高职学校和专业群建设计划。

## 🎓 专业设置

机电学院现设有 **6个专业**：

### 1. 飞行器维修技术 ⭐国家级骨干专业

**培养目标**：培养掌握飞机维修手册使用、飞机维修基本技能、飞机铆接装配与钣金成形、飞机部件与系统装调、飞机航线维护与定检等能力，面向航空航天器修理行业的高素质技术技能人才。

**主要课程**：航空维修工程英语、机械制图、航空工程材料、人为因素和航空法规、飞机结构与机械系统、飞机电气电子系统、航空发动机原理与构造、飞机维修文件及手册查询、飞机钣金成形技术、飞机铆接装配技术、飞机维护技术等。

**就业方向**：军队航空修理企业与航空兵部队、中国航空工业集团及中国航空发动机集团所属制造工厂与研究所、民用飞机和航空发动机维修公司等。

### 2. 航空发动机维修技术

**培养目标**：培养掌握航空发动机工作原理、结构组成，具备手册资料查询、发动机部附件分解装配修理试验能力，从事航空发动机装配、试车、部附件修理的高素质技术技能人才。

**主要课程**：机械制图、基本钳工技能、电工电子技术、工程热力学与气体动力学基础、航空发动机原理、航空发动机控制技术、航空发动机结构与系统、航空发动机修理技术、航空发动机试车等。

### 3. 无人机应用技术

**培养目标**：培养掌握无人机结构与系统工作原理，从事无人机植保、巡检、组装、调试与维修维护等职业岗位的复合型技术技能人才。

**主要课程**：航空维修基本技能、CATIA工程制图、无人机结构与系统、无人机飞行控制技术、无人机导航与通信技术、无人机组装调试与检修、无人机操控技术实训等。

### 4. 通用航空器维修

**培养目标**：培养掌握固定翼飞机、直升机结构与系统，具备通用航空器定检维修、运营能力的复合型技术技能人才。

**主要课程**：固定翼飞机结构与系统、直升机结构与系统、航空发动机原理与构造、通用航空器电子电气系统、通用航空器维护技术、航空发动机维护技术等。

### 5. 复合材料智能制造技术

**培养目标**：培养掌握航空复合材料零部件的成型、加工、测试和修理能力，从事航空复合材料制品工、航空器结构修理工的高素质技术技能人才。

**主要课程**：CATIA工程制图、复合材料基础、复合材料结构设计、复合材料成型工艺与设备、复合材料加工与连接技术、飞机复合材料结构修理等。

### 6. 理化测试与质检技术

**培养目标**：培养掌握各种常规无损检测技术和新型检测技术，在航空航天、石油化工、机械、冶金等行业从事关键构件无损检测的高素质技术技能型人才。

**主要课程**：计量技术、磁粉检测、渗透检测、目视检测、超声波检测、射线检测、涡流检测、无损检测专业英语等。
